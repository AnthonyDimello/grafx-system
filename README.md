# Grafx System

O **Grafx System** é um sistema desenvolvido em **Java Spring Boot** para gerenciar pedidos, produtos e serviços relacionados a uma gráfica rápida.
O projeto permite organizar e controlar informações essenciais para o funcionamento de uma empresa de impressão, facilitando o fluxo de trabalho e o gerenciamento interno.

## ✔ Funcionalidades principais

* Cadastro e gerenciamento de pedidos
* Armazenamento de informações de clientes
* Controle de produtos/serviços gráficos
* Integração com banco de dados MySQL
* Arquitetura MVC usando Spring Boot

## ✔ Tecnologias utilizadas

* **Java 21**
* **Spring Boot**
* **Spring Web**
* **Spring Data JPA**
* **MySQL**
* **Maven**

## ✔ Banco de dados

O sistema utiliza MySQL, configurado através do **Spring Data JPA**, com criação automática das tabelas quando a aplicação inicia.
